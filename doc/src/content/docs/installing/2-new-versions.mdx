---
title: Updating to new versions
---
import { Steps } from '@astrojs/starlight/components';
import { Aside } from '@astrojs/starlight/components';

<Aside type="tip">
    As a developer, you can use the script `./update.sh` to update the application. Note that local changes will be lost.
</Aside>
<Steps>

1. Update the docker image to the latest version:

    ```bash
    docker-compose pull
    docker-compose up -d
    ```

2. Run the following command to update the database schema:

    ```bash
    docker compose exec biblioteca bin/console doctrine:migration:migrate --no-interaction
    ```

3. If needed, update the typesense schema:

    ```bash
    docker-compose exec biblioteca bin/console biblioverse:typesense:populate
    ```

4. Clear the cache:

    ```bash
    docker-compose exec biblioteca bin/console cache:clear
    ```


</Steps>
